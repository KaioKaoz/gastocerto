# Changelog

## [1.0.0] - 2024-01-01

### Adicionado
- Cadastro de gastos com descrição, valor e categoria
- Listagem com filtro por categoria e total calculado
- Edição e remoção de gastos com confirmação
- Resumo financeiro agrupado por categoria
- Validações no model e no formulário
- Testes automatizados com pytest-django (models e views)
- Linting com ruff
- Pipeline CI com GitHub Actions
- Interface web com Django e templates HTML
